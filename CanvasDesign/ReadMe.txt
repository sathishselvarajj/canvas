Local Setup: Import the project as gradle project and execute gradle clean and build.

The project is developed using java jdk 1.8.
Project was developed using TDD.
Project has the following code test coverage,
1. 100% class coverage,
2. 92% method test code coverage,
3. 83% line code coverage.
